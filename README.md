# javazone.no

## Get up and running

    git clone git@github.com:javaBin/javazone.no.git
    cd javazone.no
    npm i
    
*If you're having issues, try running node=>5.1.0 & node<6.0.0*

## Development

Run in watch mode

    npm start

Build without deployment

    npm run build

Clean

    npm run clean

Deploy

    npm run deploy [env]
